# Next.js CloudRun Practice Sample

# Summary

今後、Cloud RunでWebアプリケーションを構築する際にイメージを付けるために作成したサンプルです。
TerraformでCloud Runへのデプロイを行い、Next.jsでサンプルアプリケーションを構築しています。
